Applying the brand

This section will give you guidance on how to apply the nidirect branding to your application. Before you add in any of the code below, it is important to ensure that you have included the CSS files for the nidirect brand.

These files and instructions on how to include them in your webpages and application can be found in the nidirect-UXG github repo.

Header

To apply the nidriect brand header, you can either download the full branded template with the nidirect branded header included. If you only require the code for the header it is included below.

Footer

To apply the nidriect brand footer, copy and paste the code into the appropriate section of your webpage or application.

You will need to customise the content in the footer to link to appropriate content for your application.

You will need to link to your own:

  • Privacy statement
  • Cookies
  • Accessibility statement

A link to terms and condidtions be necessary in some scenarios, but most applications will outline these in the application process.

Footer HTML code
Copy and paste the code below to insert the nidirect footer into your HTML file. 
 
  <footer class="nidirect_footer"  role="contentinfo">
   <div class="govuk-width-container">
    <p class="govuk-body">

        </li>
